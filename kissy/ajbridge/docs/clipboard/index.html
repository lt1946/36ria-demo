setData void setData(data, format, serializable)

将数据写入粘贴板

Parameters:
data <any> 理论上支持多种 格式，但js获取功能受限 ，目前仅支持 字符串。
format <String> text 目前仅支持 html 和 text 2种格式 内容。如果 flash 版本在 9.x 则仅支持 text.
serializable <Boolean> true serializable 参数确定引用和副本是否都可用，或是否只有对象的引用才可用。将 serializable 设置为 true 可使该数据对象的引用和副本都可用。将 serializable 设置为 false 则只使对象引用可用。对象引用仅在当前应用程序中有效，因此，将 serializable 设置为 false 还意味着使用该格式的数据将对其他 应用程序不可用。组件可以通过在访问该格式的数据时设置适当的剪贴板传输模式来选择获取该对象的引用或获取该对象的副本。

注意，此函数为需要在 mouseup 事件进行设置，在 mousedown 事件中执行则请确保不出现任何中断用户操作，如各种默认提示框、意外的错误等。

示例


clipboard.on("mouseUp",function(ev){
	clipboard.setData('123');
});

clearData void clearData(format)

清除指定格式内容，请注意Flash播放器版本

Parameters:
format <String> 目前仅支持 html 和 text 2种格式 内容。如果 flash 版本在 9.x 则仅支持 text.

注意，此函数为需要在 mouseup 事件进行设置，在 mousedown 事件中执行则请确保不出现任何中断用户操作，如各种默认提示框、意外的错误等。 


clipboard.on("mouseUp",function(ev){
	clipboard.clearData();
});

getData void getData(format,transferMode)

获得粘贴板内容

仅支持 Flash 播放器版本 10 以上

Parameters:
format <String> 目前仅支持 html 和 text 2种格式 内容。如果 flash 版本在 9.x 则仅支持 text.
transferMode <String> 传输模式提供有关访问剪贴板中所包含的对象时是返回一个引用还是返回一个副本的提示 目前暂不支持
